Project Explanation

Gram Jyoti monitors distributed renewable systems with remote telemetry and intelligent fault alerts. Sensor data is processed for usage trends, battery state, and anomaly detection. The mobile-facing workflow helps operators react early, improve uptime, and support sustainable energy operations in rural deployments.

Project Requirements

  • Hardware: Power/temperature sensors, controller node (ESP32/RPi), network module.
  • Software: Data ingestion script, dashboard backend, alert notification service.
  • Inputs: Sensor thresholds, telemetry schema, fault/anomaly detection rules.

Project Outcome

  • Technical: Developed a remote telemetry and anomaly-alert workflow for renewable nodes.
  • Impact: Improved reliability by enabling earlier detection of system faults.
  • Future Scope: Can integrate predictive maintenance models and larger field deployments.
