Project Explanation

This project demonstrates a compact 2-bit CPU architecture for processor fundamentals. The design includes instruction decoding, ALU operations, register updates, and clock-driven execution on FPGA hardware. It is built to make digital logic and CPU pipeline basics easier to understand in practical lab sessions.

Project Requirements

  • Hardware: FPGA development board, USB programmer, stable power supply.
  • Software: Verilog HDL toolchain (Vivado/Quartus), simulator, waveform viewer.
  • Inputs: Instruction ROM design, clock/reset setup, module-level testbench files.

Project Outcome

  • Technical: Built and validated a working 2-bit CPU flow on FPGA with modular Verilog blocks.
  • Impact: Created a practical prototype for explaining processor fundamentals in lab sessions.
  • Future Scope: Can be extended to wider instruction sets and deeper pipeline experiments.
