Project Explanation

CityPulse uses computer vision for real-time vehicle detection and adaptive traffic signal control. It prioritizes emergency vehicles and updates traffic timing dynamically to reduce congestion. ESP32 integration enables low-cost signal control and edge-triggered automation for smart-city deployment.

Project Requirements

  • Hardware: Camera feed source, ESP32 board, relay/signal control interface.
  • Software: Python, OpenCV, YOLO model weights, serial communication library.
  • Inputs: Labeled traffic classes, emergency-priority rule logic, lane mapping setup.

Project Outcome

  • Technical: Implemented AI-based vehicle detection with embedded signal control integration.
  • Impact: Demonstrated dynamic traffic optimization and emergency-priority response.
  • Future Scope: Ready for expansion with live camera feeds and multi-junction coordination.
